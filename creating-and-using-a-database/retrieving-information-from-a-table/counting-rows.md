#### 4.3.4.8 Counting Rows

数据库通常用于回答这样一个问题：“表中某种类型的数据出现的频率是多少？”例如，您可能想知道您拥有多少只宠物，或者每位拥有者拥有多少只宠物，或者您可能想要为您的动物进行各种普查操作。

统计你拥有的动物总数和“宠物表中有多少行？”是相同的问题，因为每只宠物有一个记录。 COUNT（\*）计算行数，因此计数动物的查询如下所示：

```
mysql>SELECT COUNT(*) FROM pet;
+----------+
| COUNT(*) |
+----------+
|        9 |
+----------+
```
