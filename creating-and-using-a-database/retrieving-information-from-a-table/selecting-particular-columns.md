#### 4.3.4.3 Selecting Particular Columns

如果您不想查看表格中的所有列，只需指定感兴趣的列，以逗号分隔。例如，如果您想知道您的动物出生的时间，请选择姓名和出生栏：

```
mysql>SELECT name, birth FROM pet;
+----------+------------+
| name     | birth      |
+----------+------------+
| Fluffy   | 1993-02-04 |
| Claws    | 1994-03-17 |
| Buffy    | 1989-05-13 |
| Fang     | 1990-08-27 |
| Bowser   | 1989-08-31 |
| Chirpy   | 1998-09-11 |
| Whistler | 1997-12-09 |
| Slim     | 1996-04-29 |
| Puffball | 1999-03-30 |
+----------+------------+
```
