#### 4.3.4.4 Sorting Rows

您可能在前面的示例中注意到，结果行没有按特定顺序显示。当以某种有意义的方式对行进行排序时，通常更容易检查查询输出。要对结果排序，请使用ORDER BY子句。

这里是动物生日，按日期排序：

```
mysql>SELECT name, birth FROM pet ORDER BY birth;
+----------+------------+
| name     | birth      |
+----------+------------+
| Buffy    | 1989-05-13 |
| Bowser   | 1989-08-31 |
| Fang     | 1990-08-27 |
| Fluffy   | 1993-02-04 |
| Claws    | 1994-03-17 |
| Slim     | 1996-04-29 |
| Whistler | 1997-12-09 |
| Chirpy   | 1998-09-11 |
| Puffball | 1999-03-30 |
+----------+------------+
```
